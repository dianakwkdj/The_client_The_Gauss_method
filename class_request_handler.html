Detailed Description

Класс для обработки входящих запросов (JSON и текстовых)

Класс предоставляет функциональность для обработки различных типов запросов, включая регистрацию, авторизацию, сброс пароля и решение уравнений. Наследуется от QObject для поддержки сигналов и слотов Qt.
