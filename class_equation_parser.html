Detailed Description

Класс для парсинга системы линейных уравнений.

Предоставляет статический метод для преобразования системы уравнений в матричную форму, пригодную для решения.

Парсит систему уравнений в матричную форму.

Преобразует список строковых уравнений в матрицу коэффициентов, вектор констант и список переменных.

Parameters
[in]equationsСписок строк, представляющих уравнения
[out]matrixМатрица коэффициентов (заполняется методом)
[out]constantsВектор свободных членов (заполняется методом)
[out]variablesСписок найденных переменных (заполняется методом)
[out]errorСтрока с описанием ошибки (если метод вернул false)
Returns
true - если парсинг успешен, false - если произошла ошибка
Note
Формат уравнений должен быть линейным, например: "2x + 3y = 5"
Warning
Метод не проверяет систему на совместность/определенность
