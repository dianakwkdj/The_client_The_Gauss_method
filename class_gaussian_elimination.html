Parameters
[in,out]matrixМатрица коэффициентов (N x N). После вызова содержит верхнюю треугольную матрицу
[in]constantsВектор свободных членов (длины N)
[out]solutionВектор решений (длины N)
[out]stepsСписок строк, описывающих шаги решения (для отладки/вывода)
[out]errorСтрока с описанием ошибки, если решение не найдено
[in]variablesСписок имен переменных (для генерации читаемых шагов)
Returns
true, если решение найдено, false в случае ошибки (вырожденная матрица и т.п.)

Метод выполняет прямой ход (приведение к треугольному виду) и обратный ход метода Гаусса. В процессе работы записывает текстовое описание каждого шага в steps. Если система не имеет решения или матрица вырождена, возвращает false и записывает ошибку в error.
