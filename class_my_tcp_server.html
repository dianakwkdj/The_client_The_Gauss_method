Detailed Description

Класс TCP-сервера, обрабатывающего входящие соединения и запросы

Наследуется от QTcpServer и предоставляет функциональность для:

  • Запуска сервера на указанном порту
  • Обработки входящих соединений
  • Чтения данных от клиентов
  • Отправки JSON-ответов
  • Обработки отключений клиентов
